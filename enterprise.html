How It Works

Three steps. No YAML. No configuration files. No learning curve.

1

Install the Agent

A small, lightweight process on each machine. It registers itself, reports its capabilities, and connects to the central Jeniphyr instance. One command to install.

2

Jeniphyr Knows Your Fleet

A live registry of every machine — what it can do, whether it's busy, and what it's currently running. Always up to date, always accurate.

3

Talk, Don't Configure

Type what you want in plain English. The AI figures out which machine should handle it, runs the task, and reports back. No scripts to write, no playbooks to maintain.
