Hands — What Jeniphyr Can Do

Hands are capabilities. Each Hand gives Jeniphyr the ability to interact with a specific part of the real world — files, devices, APIs, the web, and more.

Available Hands

11 built-in capabilities, with more on the way.

📁

File

Read, write, list, and search files on the server. Sandboxed to allowed directories.

"Read the config file for proofpop"

🔀

Git

Status, diffs, commit logs. See what changed and when across your repositories.

"What changed in the last 3 commits?"

💻

Shell

Run allowlisted commands only. Safe, audited, no surprises.

"How much disk space is left?"

🌐

HTTP

GET/POST to any URL. Check if sites are up, call APIs, fetch data.

"Is proofpop.test responding?"

✉️

Telegram

Send messages back to you with smart chunking for long responses.

"Every response uses this"

🧠

Memory

Store and recall facts. Shared across all assistants so they all know what you've told any of them.

"Remember: the Vizio is on 192.168.1.150"

⏰

Reminder

Set reminders with natural time support. "In 20 minutes" or "at 5pm" both work.

"Remind me at 5pm to call Shane"

🤝

Delegate

Route tasks to the right specialist assistant automatically.

"Check the server → routes to Monitor"

🔍

Search

Search the web via Brave. Get real-time answers, not stale training data.

"Search for NativePHP latest news"

📺

TV

Power, volume, input control. Works with Vizio SmartCast and Roku devices.

"Turn off the living room TV"

➕

How It Works

Four steps from your message to a real-world result.

1

You Send a Message

Open Telegram on your phone and text @Jeniphyr_bot whatever you need.

2

Jeniphyr Receives It

A webhook catches the message, drops it into a job queue, and processing begins instantly.

3

The Assistant Thinks & Acts

Claude AI reads the message, checks memory, decides which Hands to use, and executes them.

4

You Get a Response

The result comes back to Telegram. Done. You never left the chat.
